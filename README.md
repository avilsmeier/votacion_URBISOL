# Sistema de Votación URBISOL 1.0

Sistema de votación electrónica para elecciones del Concejo Directivo de la Urbanización URBISOL.

## Stack
- Node.js
- Express
- PostgreSQL
- PDFKit
- Nginx + Cloudflare

## Instalación

1. Clonar repo
2. Copiar `.env.example` a `.env`
3. Configurar variables
4. npm install
5. node create_admin.mjs
6. node src/server.js
