# Instructivo para Votantes

Guía simple para residentes, propietarios o representantes que participarán en una votación digital.

## Regla principal

El voto se emite por unidad o propiedad habilitada:

```text
1 unidad o propiedad habilitada = 1 voto
```

Si una persona representa más de una propiedad, debe registrar cada unidad por separado y recibirá un enlace distinto para cada una.

---

## 1. Registro

Ingresar al formulario de registro público indicado por el Consejo Directivo.

Completar:

- calle;
- número;
- piso/departamento si corresponde;
- nombre y apellidos;
- DNI/CE;
- teléfono;
- correo electrónico.

El correo electrónico es obligatorio porque allí se enviará el enlace personal de votación.

---

## 2. Después de registrarse

Al enviar la solicitud, el registro queda pendiente de revisión.

El Consejo Directivo revisará si la unidad está habilitada para votar.

Revisar el correo electrónico, incluyendo:

- Bandeja de entrada;
- Spam;
- Correo no deseado;
- Promociones u otras carpetas.

Se recomienda agregar el remitente oficial a contactos o remitentes seguros.

---

## 3. Solicitud aprobada

Si la solicitud es aprobada, llegará un correo con el enlace personal de votación.

Ese enlace:

- es único;
- corresponde solo a la unidad indicada;
- puede usarse una sola vez;
- no debe compartirse;
- queda inválido después de votar.

Si no encuentra el correo, debe contactar al Consejo Directivo para solicitar reemisión del enlace.

---

## 4. Solicitud rechazada

Si la solicitud es rechazada, llegará un correo indicando que fue revisada y no aprobada.

Puede incluir un motivo o nota del Consejo Directivo.

Para consultas, comunicarse con el Consejo Directivo.

---

## 5. Antes del inicio de votación

Si abre el enlace antes de la hora de inicio, el sistema indicará que la votación aún no empezó.

El enlace sigue siendo válido. Debe volver a entrar durante el horario de votación.

---

## 6. Emitir el voto

Durante el horario de votación:

1. Abrir el enlace personal recibido por correo.
2. Leer la pregunta y opciones.
3. Seleccionar una opción.
4. Presionar `Confirmar voto`.
5. Esperar la pantalla de confirmación.

Después de confirmar, el enlace ya no podrá reutilizarse.

---

## 7. Recibo de voto

Después de votar, recibirá un correo de confirmación con:

- campaña;
- unidad o propiedad;
- opción registrada;
- fecha y hora;
- código de verificación;
- enlace de validación;
- hash del voto;
- posición en cadena.

Guardar ese correo.

El recibo no permite votar otra vez ni cambiar el voto. Solo permite consultar el voto registrado.

---

## 8. Validar el voto

Puede validar el voto en:

```text
/verificar-voto
```

Debe ingresar:

- código de verificación recibido por correo;
- DNI/CE o correo electrónico usado en el registro.

El sistema mostrará el voto registrado para esa unidad.

---

## 9. Si el enlace aparece como usado o inválido

Puede ocurrir si:

- ya votó;
- está usando un enlace anterior después de una reemisión;
- copió mal el enlace;
- el enlace no corresponde a la campaña activa.

Si cree que es un error, contactar al Consejo Directivo.

---

## 10. Recomendaciones

- No compartir el enlace de votación.
- Votar dentro del horario indicado.
- Revisar Spam o Correo no deseado.
- Guardar el recibo de voto.
- Si representa más de una propiedad, registrar cada unidad por separado.

---

## Ayuda

Para dudas sobre habilitación, unidad, datos personales o reemisión de enlace, contactar al Consejo Directivo.
